About

Yuíha is a simple JavaScript API that allows you to provide custom translations for webpages. The API lets you define translation sheets, which are strings in multiple different languages associated with common ID's. You can translate any identified piece of text on the page by calling a single function. This makes Yuíha a good API to use for sites that must cater to a multilingual demographic. The simple design means you won't spend hardly any time learning how to use it, and the fact that it runs on jQuery makes it extremely versatile across browsers.




CDN

You can download the Yuíha API for use in your own web projects through its CDN. This gives you full access to Yuíha's functionality, you can create language sheets to translate any site dynamically through JavaScript. To access the API through its CDN, do the following:

<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
<script src="https://alugocp.github.io/yuiha/yuiha.js"></script>

The above code will include jQuery and subsequently Yuíha into your website. Once you've linked to these CDNs, you can begin to utilize Yuíha in your own web project.




Docs

Defining a language sheet

The first step to using Yuíha is to create a language sheet. You can do this either by defining a monolithic sheet:

yuiha.key({
 "lang":{
  "id":"text",
  "id":"text",
  ...
 },
 "lang":{
  "id":"text",
  ...
 },
 ...
});

Or incrementally by language:

yuiha.key("lang",{
 "id":"text",
 ...
});

Define what text goes where

Once you've defined your own language sheet in Yuíha, you'll want to define which strings belong where. There are two main ways to do this. The first way is to embed your IDs in html tags, like so:

<span class="yuiha id"></span>

The second way is to use the Yuíha API in a script tag:

<script>yuiha.text("id",true)</script>
<script>yuiha.text("id",false)</script>

True will create a span wrapper for your translatable content and false creates a div.


Translate your content

Yuíha API makes it easy to translate content on your page, simply run the following command to translate:

yuiha.translate("id");

And violá, that's all there is to it! You'll probably want to translate your page upon startup to some initial language, so here's how to do that:

$(document).ready(function(){
 yuiha.translate("id");
});
